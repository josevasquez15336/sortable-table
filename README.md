## Features

- Vue 3
- TypeScript
- Tailwind CSS

## Project setup and usage

Install dependencies:

```
npm install
```

Run development server:

```
npm run dev
```

Build for produciton:

```
npm run build
```

Run ESLint:

```
npm run lint
```

Lint and fix:

```
npm run lint:fix
```
